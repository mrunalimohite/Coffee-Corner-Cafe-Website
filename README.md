# ☕ Coffee Corner Cafe Website

## 📖 Project Overview

Coffee Corner Cafe is a modern and responsive cafe website built using WordPress. The website showcases the cafe's menu, gallery, company information, and contact details while providing an engaging user experience through interactive hover effects and a clean coffee-themed design.

---

## 🚀 Features

* Responsive design for desktop, tablet, and mobile devices
* Modern coffee-themed user interface
* Interactive menu cards with hover effects
* Smooth button hover animations
* Image gallery showcasing cafe ambience and products
* About Us page with company information
* Contact page with contact form
* Social media integration
* Custom footer design
* Google Maps integration (if added)
* Fast and user-friendly navigation

---

## 🛠 Technologies Used

* WordPress
* Elementor Page Builder
* Astra Theme
* Contact Form 7
* Responsive Design

---

## 🔌 Plugins Used

### Elementor

Used for designing and building pages using drag-and-drop functionality.

### Contact Form 7

Used for creating and managing the contact form.

### SVG Support

Used for uploading and displaying SVG icons and graphics.

### All-in-One WP Migration

Used for website backup and migration.

---

## 🎨 Theme Used

### Astra Theme

A lightweight and highly customizable WordPress theme used as the foundation of the website.
